## What does it?
Its a tool for having simple steps while setting up your VRChat avatar quicker. It automates stuff, so you dont need to put a VRC_AvatarDescriptor on and it tries to add lip-sync automatically. It also generates structured folders, thats helps you with keeping everything in a good structure!

Other features are things like seeing all hand-emotes in play-mode and copying contents from one avatar to another.
